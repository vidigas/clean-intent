# Extension Icons Needed

Create PNG icons in these sizes:
- icon16.png  (16x16)   - Toolbar icon
- icon48.png  (48x48)   - Extensions page
- icon128.png (128x128) - Chrome Web Store

Suggested design:
- Purple gradient background (#6366f1 to #8b5cf6)
- White layers/stack icon (like in the button)
- Rounded corners

You can use Figma, Sketch, or any design tool.
Or generate from SVG using a tool like:
  npx svg-to-png icon.svg --output icons/
